# BIT3208 Web Applications Integration Deliverables

This repository houses the complete set of practical deliverables for the Week 6 Portfolio Task. It implements robust PHP-MySQL database CRUD functionalities across three distinct management ecosystems.

## 📁 System Modules
* **/week6/** - Primary Student Form Engine & Records Grid View
* **/week6/library_system/** - Distributed Library Catalog Inventory Log Manager
* **/week6/employee_system/** - Secure Enterprise Portal with instant backend form processing and live keyword tracking lookups

## 🚀 Setup & Execution Guide
1. Copy or clone this project repository folder into your local root web directory: `C:/xampp/htdocs/BIT3208_Project/`
2. Open your web browser and navigate to phpMyAdmin: `http://localhost/phpmyadmin/`
3. Create or select a database named `studentdb` and import the structured portable deployment layout script: **`schema.sql`**
4. Test live interface operations in your web browser:
   * **Student Management:** `http://localhost/BIT3208_Project/week6/index.php`
   * **Library Catalog:** `http://localhost/BIT3208_Project/week6/library_system/index.php`
   * **Employee Workspace:** `http://localhost/BIT3208_Project/week6/employee_system/login.php`
